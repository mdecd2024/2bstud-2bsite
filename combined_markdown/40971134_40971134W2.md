# 在codespaces裡安裝Python模組
在codespaces裡的終端機輸入下面指令去安裝Python模組
<ul>
<li>chmod u+x init_replit</li>
<li>./init_replit</li>
</ul>
安裝好之後輸入 python3 main.py 就可以開啟動態網頁進行編輯
